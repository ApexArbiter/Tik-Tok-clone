# StreamVibe - Features Documentation

## Features Overview

StreamVibe is a comprehensive streaming platform with a wide range of features designed to provide users with an excellent content discovery and viewing experience.

## Core Features

### 1. Content Browsing

#### Movie Browsing
- **Movie Categories**: Browse movies by different categories
- **Trending Movies**: View currently trending movies
- **Popular Movies**: Discover popular content
- **New Releases**: See the latest movie releases
- **Genre Filtering**: Filter movies by genre (Action, Drama, Comedy, etc.)
- **Movie Details**: Comprehensive movie information pages

#### TV Series Browsing
- **Series Categories**: Browse TV series by categories
- **Trending Series**: View trending TV shows
- **Popular Series**: Discover popular series
- **New Series**: See newly released series
- **Genre Filtering**: Filter series by genre
- **Series Details**: Detailed series pages with seasons and episodes

#### Content Discovery
- **Explore Page**: Central hub for content discovery
- **Genre Pages**: Dedicated pages for each genre
- **Search Functionality**: Search across movies, series, actors, and directors
- **Recommendations**: Content recommendations based on viewing history

### 2. User Authentication

#### Registration
- **User Signup**: Create new user accounts
- **Form Validation**: Client-side and server-side validation
- **Account Creation**: Secure account creation process

#### Login

- **Cookie-based Auth**: Secure authentication using cookies

#### User Profile
- **Profile Management**: View and manage user profile

## Feature Usage Examples

### Browsing Movies
1. Navigate to `/movies`
2. Browse by category or use filters
3. Click on a movie to view details
4. Read reviews and ratings
5. Like or download the movie

### Watching Series
1. Navigate to `/series`
2. Select a series
3. Browse seasons and episodes
4. Select an episode to watch
5. Continue watching from last position

### Searching Content
1. Click search in navigation
2. Type search query
3. View real-time results
4. Filter by type (movies/series/actors)
5. Click result to view details

### Managing Subscription
1. Navigate to `/subscriptions`
2. View available plans
3. Compare plan features
4. Select a plan
5. Complete subscription process

### State Management
- User state managed in Zustand
- Component state with React hooks
- Form state with React Hook Form
